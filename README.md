# 클라우드 파일 공유 시스템 (미니 드라이브)

## 핵심 요구사항

### 파일 저장

- 문서, 이미지, PDF, 압축 파일, 영상 등 일반 업무 파일 저장 가능
- 웹 브라우저에서 파일 업로드 가능 (선택)
- 업로드한 파일은 언제든 다운로드 가능
- 파일 정보 저장
  - 파일 이름
  - 업로드 날짜
  - 파일 크기
  - 업로드한 사용자

### 폴더 관리

- 폴더 생성, 이름 변경, 삭제
- 여러 단계 폴더 구조 지원
- 삭제 파일 복구 기능 제공 (일정 기간 보관)

### 파일 공유

- 사용자끼리 파일 공유 가능
- 공유 권한 설정
  - 보기
  - 수정
  - 댓글
- 외부 공유 링크 생성 가능
- 링크 만료 기간 설정 가능

### 파일 검색

- 파일 이름으로 검색
- 조건 검색
  - 특정 날짜 이후 업로드된 파일
  - 특정 사용자가 업로드한 파일
  - 특정 파일 형식
- 검색 결과는 빠르게 표시

### 파일 버전 관리

- 파일 수정 시 이전 버전 저장
- 이전 버전 목록 확인
- 이전 버전 다운로드
- 이전 버전으로 복원
- 버전은 무제한이 아니라 일정 개수/기간 보관

### 사용자 계정

- 이메일/비밀번호 로그인
- 관리자 기능
  - 사용자 계정 생성/삭제
  - 사용자별 저장 공간 설정
- 사용자 그룹 기능 추가 (선택)

## 보안 요구사항

- 로그인한 사용자만 시스템 사용 가능
- 사용자마다 허용된 기능만 사용 가능
- 파일 전송 시 암호화 적용
- 외부 공유 링크는 만료 기간으로 접근 제한

## 성능 요구사항

- 약 200명 규모에서도 안정적으로 동작
- 특히 파일 검색과 다운로드가 빨라야 함

## 향후 확장 기능

- 향후 필요에 따라 기능을 확장할 수 있도록 설계
